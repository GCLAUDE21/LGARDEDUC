# L Gard'Educ

Site web fullstack pour une éducatrice canine, permettant la présentation des services et la gestion des réservations.

## Stack

- React / Vite / SCSS (frontend)
- Node.js / Express (backend)
- MongoDB / Mongoose (base de données)
- JWT / bcrypt (authentification)

## Fonctionnalités

- Présentation des services de l'éducatrice
- Inscription et connexion avec authentification JWT
- Routes protégées
- Interface responsive
